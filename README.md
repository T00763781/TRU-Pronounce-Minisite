# TRU Name Inclusion Minisite (Rebuild)

This rebuild fixes the content loader by parsing `content.md` in KEY/BODY format (no external libraries).
It also uses a TRU-like header + banner layout inspired by the current tru.ca structure.

## Files
- `index.html` page layout
- `styles.css` TRU-like styling
- `site.js` loads `content.md`, embeds the tool, and powers the carousel
- `content.md` longform copy (edit this)

## Deploy
Push to GitHub and enable GitHub Pages from `main` / root.
