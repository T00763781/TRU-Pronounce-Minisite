Name inclusion

Learning and using someone’s name the way they prefer is a practical, everyday way to show respect. This tab includes guidance plus a pronunciation helper you can use as a starting point.

Curiosity over guessing Practice over performance Consent over convenience Repair over embarrassment

Pronunciation helper

Starting point only: adjust to match the person’s preference. When in doubt, ask the person — they’re the source of truth.

Trouble loading? Open the helper in a new tab.

How to learn a name

  1. Ask: “Could you help me pronounce your name the way you prefer?”
  2. Repeat it back and confirm.
  3. Write a quick note (phonetic + cue).
  4. If you miss it: brief apology, correct, continue.

How to teach your name

  1. Say it slowly, then at normal speed.
  2. Share a syllable breakdown.
  3. Add a cue (“rhymes with…”, “like…”).
  4. Invite correction.
