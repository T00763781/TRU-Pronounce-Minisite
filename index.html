How to learn a name

  1. Ask: “Could you help me pronounce your name the way you prefer?”
  2. Repeat it back and confirm.
  3. Write a quick note (phonetic + cue).
  4. If you miss it: brief apology, correct, continue.

How to teach your name

  1. Say it slowly, then at normal speed.
  2. Share a syllable breakdown.
  3. Add a cue (“rhymes with…”, “like…”).
  4. Invite correction.
